# Hangman Game

## Description
This is a simple text-based Hangman game built using Python.

The program randomly selects a word, and the player guesses one letter at a time. The player has six incorrect attempts before losing the game.

## Features
- Random word selection
- Six incorrect guesses allowed
- Prevents duplicate guesses
- Input validation
- Win and lose messages

## Concepts Used
- Python Lists
- Strings
- Loops
- Conditionals
- Random Module
- User Input

## How to Run

1. Install Python 3.
2. Open a terminal.
3. Run:

```bash
python hangman.py
```

Example

```
WELCOME TO HANGMAN

Word: _ _ _ _ _ _

Remaining Attempts: 6

Enter a letter: p

Correct!

Word: p _ _ _ _ _
```
