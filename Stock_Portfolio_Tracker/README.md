# Stock Portfolio Tracker

## Description

A simple Python application that calculates the total value of a stock portfolio using predefined stock prices.

## Features

- User enters stock symbols and quantities.
- Uses predefined stock prices.
- Calculates total investment value.
- Saves results to a CSV file.
- Handles invalid stock symbols and invalid input.

## Technologies

- Python 3
- CSV Module

## How to Run

```bash
python portfolio.py
```

## Example

```
Enter stock symbol: AAPL
Enter quantity: 5

Enter stock symbol: TSLA
Enter quantity: 2

Total Investment Value: $1400
Portfolio saved successfully.
```
