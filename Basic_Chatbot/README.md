# Basic Chatbot

# Description

A simple rule-based chatbot developed in Python. The chatbot responds to predefined user inputs and demonstrates the use of basic programming concepts.

# Features

- Greets the user
- Answers simple questions
- Displays the current date
- Displays the current time
- Help menu
- Exit command

# Concepts Used

- Functions
- Loops
- If-elif statements
- User input
- Date and Time module

# Requirements

- Python 3.x

# How to Run

```bash
python chatbot.py
```

# Example

```
You: hello
Bot: Hello! Nice to meet you.

You: time
Bot: The current time is 14:35:10

You: bye
Bot: Goodbye! Have a great day.
```
